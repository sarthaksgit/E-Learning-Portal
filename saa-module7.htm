Amazon Elastic Map Reduce (EMR) and Amazon ElastiCache

Duration: 10 Hrs

Amazon Elastic Map Reduce Introduction

Explore the fundamentals of Amazon Elastic Map Reduce (EMR), a cloud-based big data platform that simplifies the processing and analysis of large datasets using popular frameworks such as Apache Hadoop and Apache Spark.

AWS EMR - Clusters, Nodes, and Deployment in an AZ

Learn about EMR clusters and nodes, and understand how to deploy EMR in a specific Availability Zone (AZ) to optimize performance and availability.

Amazon ElastiCache Introduction

Introduction to Amazon ElastiCache, a fully managed, in-memory caching service that provides a high-performance, scalable caching solution for improving application response times.
